#### 占坑

#### 可尝试安装使用
```code
npm i tiga-cli -g
```

#### 新建项目
```bash
tiga init test-cli
```

#### 配置好信息之后下载模版

#### 下载模版完成之后进入项目
安装依赖
```bash
npm i
```

运行项目
```bash
npm run dev
```
